# نظام إدارة العقارات الخاص

تطبيق PWA محلي لإدارة العقارات. الكود فقط يرفع على GitHub، أما بيانات العقارات والصور والأرقام تحفظ داخل جهازك عبر IndexedDB.

## التشغيل
```bash
npm install
npm run dev
```

## البناء للرفع على GitHub Pages
```bash
npm run build
```
ارفع محتويات مجلد `dist` أو استخدم إعدادات GitHub Pages حسب طريقتك.

## الخصوصية
- لا يوجد Firebase.
- لا يوجد سيرفر خارجي للبيانات.
- النسخ الاحتياطي من الإعدادات يصدر ملف `.aqarbackup` يحتوي كل البيانات والصور والشعارات.

## التثبيت على الجوال
افتح رابط GitHub Pages من Safari/Chrome ثم اختر: إضافة إلى الشاشة الرئيسية.
